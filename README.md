# Agenda API REST
API REST de um pequeno sistema de agenda desenvolvido utilizando Spring Boot. Trata-se da prova para seleção ao cargo de Desenvolvedor no Instituto Creathus.

# Tecnologias utilizadas
- JDK 15
- Spring Boot 2.4.1
- Maven
- MySQL

# Ferramentas utilizadas
- IntelliJ IDEA
- DBeaver Community
- MySQL Community
- Postman
